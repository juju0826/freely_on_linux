# freely_on_linux

Running free software on Linux in a way that is stable, predictable, and sustainable.

在 Linux 上，把自由軟體接成一個可以長期安心使用的系統。

---

## 這個 repository 在做什麼？

這裡不是快速安裝指南，也不是設定參數的堆疊。

這個 repository 記錄的是一條**實際走過、已在真實系統中長期運作**的路線：
當我們在 Linux 上自架服務時，  
如何讓自由軟體不是「勉強能用」，而是「結構正確、行為可預期」。

很多時候，系統沒有壞，但就是讓人不安心：
- 有時順，有時卡
- 背景工作在錯的時間跑
- 多個服務彼此干擾
- 管理介面出現警告，卻難以判斷哪些真的重要

這個 repo 嘗試回答的不是「怎麼裝」，而是：

> **為什麼系統會不穩？  
又該把問題放在哪一層解決？**

---
